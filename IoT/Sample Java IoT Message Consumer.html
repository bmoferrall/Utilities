Introduction

The Java program described here can be used to subscribe to a topic in the Watson IoT internal message subsystem. It will simply print out the topic name and message payload. It was created as a proof of concept for the LOGICALINTERFACE data source type described here that could be adapted for use in IOC.

Setup

For the purposes of the examples shown here devices and interfaces are set up according to this page. The simulator describe here is used to publish messages from the sample device. A sample rule has been set up (link) for the following condition: $state.category = 'Fire'.
