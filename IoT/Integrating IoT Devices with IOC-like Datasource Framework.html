The attached documents describe a method of integrating events emitted by WIoT-registered devices into an IOC-like datasource framework.

For this purpose we define a new LOGICALINTERFACE data source type:

  1. LOGICALINTERFACE datasource type - integrating with IOC's existing datasource framework (doc)
  2. LOGICALINTERFACE datasource type - implementing as part of a new datasource framework for the IoT platform (doc)
